# Prompt pre konverziu bio záhrady do RAG knowledge dokumentu

Prosím, na základe všetkých informácií o bio záhrade ktoré sme spoločne vypracovali v tomto chate, vytvor jeden komplexný markdown dokument, ktorý bude optimalizovaný pre použitie v RAG (Retrieval Augmented Generation) systéme.

## RAG optimalizácia:

**Dokument bude slúžiť ako knowledge base pre AI systém**, takže je potrebné:

- **Štruktúrovať informácie do jasných sekcií** s opisnými nadpismi, ktoré umožnia efektívne vyhľadávanie
- **Každá sekcia má byť self-contained** - obsahovať kompletné informácie o danej téme
- **Používať kľúčové slová a synonymá** ktoré môžu byť predmetom dotazov (napr. "siatba", "sadenie", "pestovanie", "termíny")
- **Zahrňte všetky praktické detaily** - dátumy, postup, požiadavky, tipy
- **Vytvorte cross-referencie** medzi súvisiacimi témami

## Technické požiadavky:

- **Markdown formát** s jasnou hierarchiou nadpisov (H1, H2, H3)
- **Tabuľky pre štruktúrované dáta** (harmonogramy, rotácie, charakteristiky rastlín)
- **Konzistentné pomenovanie** - používajte rovnaké termíny pre rovnaké veci
- **Úplné informácie** - zahrňte všetko čo sme diskutovali, pretože RAG systém bude z tohto dokumentu čerpať odpovede

## Cieľ:
Vytvor jeden markdown súbor ktorý môžem priamo nahrať do nášho RAG systému a ktorý bude slúžiť ako kompletná knowledge base pre všetky otázky týkajúce sa nášej bio záhrady.
