## RIEŠENIE B: PORT FORWARDING

Ak DMZ nie je možné, nastaviť tieto pravidlá:

### DÔLEŽITÉ: Cieľový port = Externý port!

```
SPRÁVNE:   9443 → 192.168.2.10:9443 ✅
NESPRÁVNE: 9443 → 192.168.2.10:443  ❌
```
